Forge Web Starter

Welcome to the Forge Web Starter project! This is a Spring Boot learning application that demonstrates REST APIs, WebSocket communication, and game state management.

Open Test Client View Cards API

🚀 Quick Start

  • Click "Open Test Client" to test all features
  • Create a game and connect via WebSocket
  • Try drawing cards and playing them
  • Watch real-time updates in action!

📚 Learning Resources

  • REST API: /api/cards and /api/game
  • WebSocket: ws://localhost:8080/ws/game/{'{gameId}'}
  • Documentation: Check the README.md file
