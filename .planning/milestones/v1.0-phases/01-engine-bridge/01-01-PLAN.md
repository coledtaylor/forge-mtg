   - Parent: `<groupId>forge</groupId> <artifactId>forge</artifactId> <version>${revision}</version>`
   - ArtifactId: `forge-gui-web`

     - `forge-gui` (scope: compile) with exclusion of `org.eclipse.jetty:*` (wildcard)

3. **WebGuiBase.java** (package `forge.web`): Implement IGuiBase for headless web context.
   - Use `HeadlessGuiDesktop.java` as reference -- WebGuiBase is similar but adds the pseudo-EDT executor.
   - Create a `private final ExecutorService edtExecutor = Executors.newSingleThreadExecutor(r -> { Thread t = new Thread(r, "Web-EDT"); t.setDaemon(true); return t; });`
   - `isGuiThread()`: return `Thread.currentThread().getName().equals("Web-EDT")`
   - `invokeInEdtLater(Runnable proc)`: `edtExecutor.submit(proc);`
   - `invokeInEdtAndWait(Runnable proc)`: if `isGuiThread()` then `proc.run()` else `edtExecutor.submit(proc).get()` (catch ExecutionException/InterruptedException, wrap in RuntimeException)
   - `invokeInEdtNow(Runnable proc)`: `proc.run();`
   - `isRunningOnDesktop()`: return `false`
   - `getAssetsDir()`: accept assets dir path as constructor parameter (String), return it. Default to `"."` for tests.
   - For all GUI dialog methods (showBugReportDialog, showErrorDialog, showImageDialog, etc.): log the message via SLF4J Logger and return no-op/null/empty as appropriate (match HeadlessGuiDesktop pattern)
   - For `getCurrentVersion()`: return "forge-gui-web"
   - Add a `shutdown()` method that calls `edtExecutor.shutdownNow()`

4. **WebGuiBaseTest.java** (package `forge.web`, TestNG):
   - Test `isGuiThread()` returns false on the test thread
   - Test `invokeInEdtLater()` executes the runnable on the "Web-EDT" thread (use CountDownLatch + thread name capture)
   - Test `invokeInEdtAndWait()` executes synchronously and captures thread name as "Web-EDT"
   - Test `invokeInEdtNow()` runs on the calling thread (not Web-EDT)
   - Test `isRunningOnDesktop()` returns false
   - Test `getAssetsDir()` returns the constructor-provided path
   - Use `@AfterClass` to call `shutdown()` on the WebGuiBase instance
