1. **WebGuiGame.java** (package `forge.web`): Full IGuiGame implementation extending AbstractGuiGame.

   **Constructor:**
   - Parameters: `WsContext wsContext, ObjectMapper objectMapper, WebInputBridge inputBridge, ViewRegistry viewRegistry`
   - Store all as private final fields
   - `private final AtomicLong sequenceCounter = new AtomicLong(0);`
   - `private static final Logger LOG = LoggerFactory.getLogger(WebGuiGame.class);`
   - `private static final long INPUT_TIMEOUT_MINUTES = 5;`

   **Core send/sendAndWait methods (private):**
   ```java
   private void send(MessageType type, Object payload) {
       OutboundMessage msg = new OutboundMessage(type, null, sequenceCounter.incrementAndGet(), payload);
       String json = objectMapper.writeValueAsString(msg);
       wsContext.send(json);
   }

   private void send(MessageType type, String inputId, Object payload) {
       OutboundMessage msg = new OutboundMessage(type, inputId, sequenceCounter.incrementAndGet(), payload);
       String json = objectMapper.writeValueAsString(msg);
       wsContext.send(json);
   }

   private <T> T sendAndWait(MessageType type, Object payload, Class<T> responseType) {
       String inputId = UUID.randomUUID().toString();
       CompletableFuture<String> future = inputBridge.register(inputId);
       send(type, inputId, payload);
       try {
           String rawResponse = future.get(INPUT_TIMEOUT_MINUTES, TimeUnit.MINUTES);
           return objectMapper.readValue(rawResponse, responseType);
       } catch (TimeoutException e) {
           inputBridge.complete(inputId, null); // cleanup
           throw new GameSessionExpiredException("Input timeout after " + INPUT_TIMEOUT_MINUTES + " minutes");
       } catch (Exception e) {
           throw new RuntimeException("Input bridge error", e);
       }
   }
   ```
