   - Props: `{ card: CardDto; rotation: number; translateY: number; index: number; isPlayable?: boolean; onHoverEnter: (cardName: string, e: React.MouseEvent) => void; onHoverMove: (e: React.MouseEvent) => void; onHoverLeave: () => void; onDoubleClick?: (cardId: number) => void }`
   - Card size: 100px wide x 140px tall (aspect-[5/7])
   - Default state: apply rotation and translateY transforms. Card uses GameCardImage at 100px width.
   - Hover state: on mouseenter, raise card 40px up (`translateY: -40px` overriding the arc translateY), scale to 1.1x, z-index-10 above siblings, remove arc rotation. Transition: `transition-transform duration-150 ease-out`.
   - Playable indicator: if `isPlayable` is true, show subtle accent ring: `ring-1 ring-primary/40`
   - Mouse handlers: onMouseEnter calls onHoverEnter(card.name, e), onMouseMove calls onHoverMove(e), onMouseLeave restores default transform and calls onHoverLeave()
   - Double-click: calls onDoubleClick(card.id) if provided. Card fades from hand briefly: `opacity-50 transition-opacity duration-200` on click (optimistic visual feedback before engine processes)
   - Single click: just selects/previews (handled by hover). No additional click handler needed.
   - Manage hover state with local useState for `isHovered`

   - Tapped cards need adjusted container dimensions to avoid layout issues: when tapped, the visual width becomes the height and vice versa. Wrap in a container that has the tapped dimensions.

   - Render GameCardImage with card.name at the specified width

   - Position: absolute bottom-1 right-1, stacked vertically if multiple counter types
