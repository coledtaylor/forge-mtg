# Phase 3 — UI Design Contract

> Visual and interaction contract for the deck builder. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Phase scope:** Full deck building experience -- search cards, add/remove with quantity controls, list and grid views, statistics charts, format validation, sideboard management, commander slot, and basic land bar. Builds on Phase 2 scaffold (SearchBar, CardGrid, CardImage, shadcn components, dark theme).

Inherited from Phase 2. No changes.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, mana symbol gaps, +/- button internal padding |
| sm | 8px | Card row internal padding, compact chart element spacing |
| md | 16px | Default element spacing, panel padding, grid gap |
| lg | 24px | Section padding, tab content padding, panel header padding |
| xl | 32px | Gap between left and right panels in split layout |
| 2xl | 48px | Not used in Phase 3 |
| 3xl | 64px | Not used in Phase 3 |

Exceptions:
- Hover preview card image renders at 260px wide (natural aspect ratio height ~362px)
- Mini stats bar in deck tab is 80px tall
- Touch target for +/- quantity buttons: minimum 32px square (8-point aligned)
- Commander slot: 120px wide card image (natural height ~167px)

## Typography

Inherited from Phase 2. No changes.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 600 (semibold) | 1.2 |

**Font weights used:** 400 (regular) and 600 (semibold). No other weights permitted.

**Phase 3 typography mapping:**
- Card name in deck list row: Body (14px, 400)
- Card quantity in deck list row: Body (14px, 600)
- Mana cost symbols: rendered via mana-font at 14px (inherits from row font size)
- Card type group headers (Creatures, Instants, etc.): Label (12px, 400), uppercase, muted-foreground
- Tab labels ("Deck", "Stats", "Sideboard"): Body (14px, 600) when active, 400 when inactive
- Chart axis labels: Label (12px, 400)
- Chart data labels: Label (12px, 400)
- Deck total card count: Heading (20px, 600)
- Stats panel section headings: Body (14px, 600)
- Format validation badge text: Label (12px, 400)
- Commander slot label: Label (12px, 400), uppercase
- Basic land bar labels: Label (12px, 400)

```
|  +------------------------------------------------------+ |

```

### Deck Editor View (main Phase 3 surface)

Two-column split layout. Fixed 50/50 at 1280px minimum viewport (~640px each).

**Focal point:** The grouped deck list in the right panel (DeckPanel, Deck tab) is the primary focal point. It occupies the dominant eye-tracking position (right panel, vertically centered) and is where the user spends most interaction time adding, removing, and reviewing cards. The search panel (left) is secondary -- it feeds cards into the focal deck list.

```
+-----------------------------+-------------------------------+
| LEFT PANEL (50%)            | RIGHT PANEL (50%)             |
| Card Search                 | Deck Contents                 |
|                             |                               |
| [Search input......] [Go]  | Commander: [Atraxa, Grand...] |
| [Color v] [Type v] [CMC]   | [Deck] [Stats] [Sideboard]    |
|                             | [List|Grid]     72/100 cards  |
| +-----+ +-----+ +-----+   |                               |
| |Card | |Card | |Card |   | -- Creatures (24) ----------  |
| |     | |Card | |     |   | 4x  Lightning Angel  {2}{R}  |
| +-----+ +-----+ +-----+   | 3x  Snapcaster Mage  {1}{U}  |
| +-----+ +-----+ +-----+   |                               |
| |Card | |Card | |Card |   | -- Instants (12) -----------  |
| |     | |     | |     |   | 4x  Lightning Bolt     {R}   |
| +-----+ +-----+ +-----+   | 4x  Counterspell      {U}{U} |
|                             |                               |
| Page 1 of 20               | [Mini mana curve] [Color dots]|
|                             |                               |
|                             | [Plains+/-] [Island+/-] ...   |
+-----------------------------+-------------------------------+
```

**Split layout rules:**
- Gap between panels: xl (32px)
- Each panel: independent vertical scroll
- Left panel max content: SearchBar + CardGrid + PaginationBar
- Right panel: CommanderSlot (if Commander format) + Tabs + content + MiniStats + BasicLandBar
- Minimum viewport: 1280px (per REQUIREMENTS.md)

## Chart Specifications

### Mana Curve (bar chart)

- **Type:** Custom SVG vertical bar chart
- **Data:** 8 bars for CMC 0, 1, 2, 3, 4, 5, 6, 7+
- **Bar width:** 32px, gap: 4px, total width: 284px
- **Height:** 120px chart area + 20px axis labels = 140px total
- **Bar fill:** `var(--primary)` (accent blue)
- **Bar border-radius:** 2px top corners
- **Axis labels:** CMC values below each bar (Label, 12px, muted-foreground)
- **Data labels:** Card count above each bar when > 0 (Label, 12px, foreground)
- **Mini version (in deck tab):** 80px total height, 16px bar width, 2px gap, no data labels

### Color Distribution (donut chart)

- **Type:** Custom SVG donut chart
- **Data:** 6 segments max (W, U, B, R, G, Colorless)
- **Diameter:** 160px outer, 80px inner (donut hole)
- **Segment colors:** MTG color hex values from color reference table
- **Black segment stroke:** 1px `var(--border)` for visibility against dark background
- **Center text:** Total non-land card count (Heading, 20px, 600)
- **Legend:** Below chart, horizontal row of color dots (12px) + labels (Label, 12px)
- **Empty state:** Single gray ring with "No cards" center text

### Card Type Breakdown (horizontal bar chart)

- **Type:** Custom SVG horizontal bar chart
- **Data:** Up to 7 rows (Creatures, Instants, Sorceries, Enchantments, Artifacts, Planeswalkers, Lands)
- **Row height:** 24px with 4px gap
- **Label width:** 100px left-aligned (Label, 12px, muted-foreground)
- **Bar fill:** `var(--primary)` (accent blue)
- **Bar border-radius:** 2px right corners
- **Count:** Right of bar (Label, 12px, foreground), 8px gap from bar end
- **Zero-count rows:** Hidden (only show types present in deck)
