# Phase 6 — UI Design Contract

> Visual and interaction contract for deck import and export modals. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> **Phase scope reminder:** This phase adds two modal dialogs (import and export) to the existing deck editor, plus a toast notification system. No changes to the deck editor layout, card search, or statistics. All parsing and formatting is server-side.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | shadcn (already initialized) |
| Preset | base-nova, dark mode, neutral base color |
| Component library | base-ui (via shadcn/ui) |
| Icon library | lucide-react |
| Font | Inter (system font stack fallback: `ui-sans-serif, system-ui, sans-serif`) |

**Existing assets reused from Phases 2-5:**
- shadcn `dialog`, `button`, `select` components
- `useDeckEditor()` hook for deck state management
- DeckPanel header bar with existing icon buttons (Back, Play, List/Grid toggle)
- All CSS variables in `index.css` (dark theme established)

**New dependency:**
- `sonner` -- toast notification library (shadcn-recommended, installed via `npm install sonner`)

## Typography

Inherited from Phases 2-5. No changes.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 400 (regular) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |

**Font weights used:** 400 (regular) and 600 (semibold). No other weights permitted.

**Phase 6 typography mapping:**
- Dialog titles ("Import Deck", "Export Deck"): Heading (20px, 600)
- Textarea input text: Body (14px, 400), `font-family: monospace`
- Export preview textarea: Body (14px, 400), `font-family: monospace`
- Parse preview token lines: Body (14px, 400)
- Summary counts text: Label (12px, 400)
- File drop zone text: Body (14px, 400), muted-foreground
- Format dropdown labels: Body (14px, 400)
- Import/Export header button tooltips: Label (12px, 400)
- Toast text: Body (14px, 400)

## Color

Inherited from Phase 2 dark theme. All values reference existing CSS variables in `index.css`. No new CSS variables added.

| Role | CSS Variable | oklch Value | Usage |
|------|-------------|-------------|-------|
| Dominant (60%) | `--background` | `oklch(0.15 0.005 285)` | Page background behind modal overlay |
| Secondary (30%) | `--card` | `oklch(0.25 0.003 285)` | Modal dialog surface, textarea background |
| Accent (10%) | `--primary` | `oklch(0.637 0.194 255)` | See reserved list below |
| Destructive | `--destructive` | `oklch(0.45 0.15 25)` | Unknown/unsupported card indicators in preview |
| Muted text | `--muted-foreground` | `oklch(0.708 0.003 285)` | File drop zone text, summary counts, comment/section lines |
| Border | `--border` | `oklch(0.25 0.003 285)` | Textarea border, file drop zone dashed border, preview divider |

**Accent reserved for:**
- "Replace Deck" button (primary variant) -- the primary import CTA
- "Copy to Clipboard" button (primary variant) -- the primary export CTA
- Import/Export icon buttons in DeckPanel header (ghost variant, accent on hover)
- Focus ring on textarea, file upload area, and format dropdown

**NOT accent:** Token line text, summary count numbers, file drop zone icon, export preview text, format dropdown text.

**Parse token colors (semantic, Phase 6 specific):**

| Token Type | Color | Indicator |
|------------|-------|-----------|
| LEGAL_CARD | `oklch(0.55 0.15 145)` (green) | Check icon |
| LIMITED_CARD, CARD_FROM_NOT_ALLOWED_SET | `oklch(0.65 0.15 85)` (yellow/amber) | Warning triangle icon |
| UNKNOWN_CARD, UNSUPPORTED_CARD | `var(--destructive)` | X icon |
| COMMENT, DECK_SECTION_NAME | `var(--muted-foreground)` | No icon |

These token colors are applied as inline text color on each preview line. They use Tailwind utility classes:
- Green: `text-green-500`
- Amber: `text-amber-500`
- Red: `text-destructive`
- Gray: `text-muted-foreground`

## Layout Contract

### Import/Export Button Placement (DeckPanel Header)

Two new icon buttons added to the DeckPanel header bar, positioned between the "Play This Deck" button and the List/Grid view toggle.

```
+---------------------------------------------------------------+
| [<- Back]  Deck Name  [Saved]   [Import] [Export] [Play] [L|G] |
+---------------------------------------------------------------+
```

| Property | Value |
|----------|-------|
| Button variant | `ghost` |
| Button size | `icon` (32x32px) |
| Import icon | `Upload` (lucide-react) |
| Export icon | `Download` (lucide-react) |
| Icon size | 16px |
| Gap between buttons | 4px (xs token) |
| Tooltip | "Import Deck" / "Export Deck" |

### Import Modal Layout

Two-column layout: input on left, live preview on right. Modal width overrides default shadcn DialogContent.

```
+-------------------------------------------------------------------+
| Import Deck                                              [X]       |
+-------------------------------------------------------------------+
| LEFT COLUMN (50%)           | RIGHT COLUMN (50%)                   |
|                             |                                       |
| +-------------------------+ | Preview                               |
| | Drop .dck/.dec/.txt     | |                                       |
| | file or click to browse | | [check] 4 Lightning Bolt              |
| +-------------------------+ | [check] 4 Counterspell                |
|                             | [warn]  2 Sol Ring (restricted)       |
| +-------------------------+ | [x]     1 Fjdkslfjdsk                 |
| | Paste deck list here... | | [gray]  // My Deck                    |
| |                         | | [gray]  Sideboard                     |
| | 4 Lightning Bolt        | | [check] 2 Rest in Peace               |
| | 4 Counterspell          | |                                       |
| | 2 Sol Ring              | |                                       |
| | 1 Fjdkslfjdsk           | |                                       |
| | // My Deck              | |                                       |
| | Sideboard               | |                                       |
| | 2 Rest in Peace         | | ------------------------------------ |
| |                         | | 10 recognized, 1 warning, 1 not found |
| +-------------------------+ |                                       |
+-------------------------------------------------------------------+
| [Replace Deck]                              [Add to Deck] [Cancel] |
+-------------------------------------------------------------------+
```

| Property | Value |
|----------|-------|
| Dialog max-width | `sm:max-w-3xl` (768px) |
| Dialog max-height | `max-h-[80vh]` |
| Content padding | 24px (lg token) |
| Column gap | 32px (xl token) |
| Column layout | `grid grid-cols-2` |
| Minimum content height | 400px (`min-h-[400px]`) |

### File Upload Drop Zone

| Property | Value |
|----------|-------|

| Property | Value |
|----------|-------|
| Scroll | `overflow-y-auto` |
| Section label | "Preview" in Label (12px, 400), muted-foreground, uppercase |
| Token line height | 28px per line (comfortable reading) |
| Token line padding | 4px vertical, 8px horizontal |
| Icon size | 14px, inline before card text |
| Icon-to-text gap | 8px (sm token) |
| Summary divider | 1px `var(--border)` horizontal rule above summary |
| Summary position | Sticky bottom of preview panel |
| Summary text | Label (12px, 400), muted-foreground |

## Interaction States

### Import/Export Header Buttons

| State | Visual |
|-------|--------|
| Default | Ghost button, foreground icon |
| Hover | `bg-accent/10`, tooltip appears after 500ms |
| Active | `bg-accent/20` |
| Focus | Focus ring (accent) |

### File Drop Zone

| State | Visual |
|-------|--------|
| Default | 2px dashed border, muted-foreground text + icon |
| Hover | Border color `var(--primary)`, cursor pointer |
| Drag over | Border color `var(--primary)`, `bg-primary/5`, text "Drop file to import" |
| File loaded | Border solid 1px accent, shows filename + file size, "Clear" text button |

### Import Textarea

| State | Visual |
|-------|--------|
| Empty | Placeholder text "Paste deck list here..." in muted-foreground |
| Has content | Monospace text, foreground color |
| Focus | Focus ring (accent) |
| File loaded | Textarea populated with file contents, editable |

### Preview Panel

| State | Visual |
|-------|--------|
| Empty (no input) | Muted text: "Paste or upload a deck list to see preview" |
| Parsing | Subtle loading indicator (opacity pulse on previous results) |
| Parsed results | Color-coded token lines with icons and summary |
| All recognized | Summary in default foreground: "10 recognized" |
| With warnings | Summary includes amber count: "8 recognized, 2 warnings" |
| With unknowns | Summary includes red count: "8 recognized, 1 warning, 1 not found" |

### Import Action Buttons

| State | Visual |
|-------|--------|
| Replace Deck (default) | Primary variant, accent background, "Replace Deck" label |
| Add to Deck (default) | Outline variant, "Add to Deck" label |
| No parsed cards | Both buttons disabled, `opacity-50`, `cursor-not-allowed` |
| With unknowns | Buttons enabled, label appends skip count: "Replace Deck (1 skipped)" |
| Importing | Active button shows spinner + "Importing...", disabled |
| Import complete | Dialog closes, toast: "Deck imported successfully" |

### Export Format Dropdown

| State | Visual |
|-------|--------|
| Default | Shows "Generic text" (default format selected) |
| Open | Four options: Generic text, MTGO, Arena, Forge .dck |
| Changed | Preview textarea updates to show formatted text in selected format |
| Loading | Preview textarea shows skeleton lines while export text loads |

### Copy to Clipboard Button

| State | Visual |
|-------|--------|
| Default | Primary variant, "Copy to Clipboard" with `Copy` icon (lucide) |
| Hover | Accent background brightens slightly |
| Clicked | Button briefly shows "Copied!" with `Check` icon (150ms), then reverts |
| Toast | sonner success toast: "Copied to clipboard" appears bottom-right |
| Clipboard error | sonner error toast: "Could not copy. Try selecting the text manually." |
