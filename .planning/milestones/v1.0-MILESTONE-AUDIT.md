## Integration Issues

### Issue 1: Format Validation for Casual 60-card and Jumpstart (Medium)
**Affected:** DECK-10
**Impact:** `FormatValidationHandler` returns HTTP 400 for these format strings because `FModel.getFormats().get("Casual 60-card")` returns null. Stats panel silently shows no validation result. Not a crash.
**Fix:** Map frontend format strings to Forge format names, or return empty (always-legal) result for unknown formats.

### Issue 2: Duplicate GameStartConfig Type (Low)
**Affected:** No requirements
**Impact:** `GameStartConfig` defined identically in `useGameWebSocket.ts` and `GameLobby.tsx`. Structural typing makes this safe today, but definitions could drift.
**Fix:** Consolidate to `src/types/game.ts`.

### Issue 3: AI Deck Selection Fallback (Low)
**Affected:** SETUP-03 (degraded UX, not broken)
**Impact:** Standard/Jumpstart games may find no AI decks matching format, falling back to 60 Forests.
**Fix:** Content/seeding issue — ensure AI decks are tagged with format names.
