1. **Update WebServer.java** to add full WebSocket endpoint and game session management:

   **Session management:**
   - Add `private static final ConcurrentHashMap<String, GameSession> activeSessions = new ConcurrentHashMap<>();`
   - Create inner class `GameSession` with fields: `HostedMatch hostedMatch, WebGuiGame webGuiGame, WebInputBridge inputBridge, ViewRegistry viewRegistry, WsContext wsContext`

   **WebSocket endpoint** (`/ws/game/{gameId}`):
   ```java
   app.ws("/ws/game/{gameId}", ws -> {
       ws.onConnect(ctx -> {
           String gameId = ctx.pathParam("gameId");
           LOG.info("WebSocket connected for game: {}", gameId);
           ctx.enableAutomaticPings();
           // Don't start game yet -- wait for START_GAME message
           // Store ctx temporarily for session creation
       });

       ws.onMessage(ctx -> {
           InboundMessage msg = objectMapper.readValue(ctx.message(), InboundMessage.class);
           String gameId = ctx.pathParam("gameId");

           switch (msg.getType()) {
               case START_GAME -> handleStartGame(ctx, gameId, msg);
               case CHOICE_RESPONSE, CONFIRM_RESPONSE, AMOUNT_RESPONSE -> {
                   GameSession session = activeSessions.get(gameId);
                   if (session != null) {
                       String responseJson = objectMapper.writeValueAsString(msg.getPayload());
                       boolean completed = session.inputBridge.complete(msg.getInputId(), responseJson);
                       if (!completed) {
                           LOG.warn("No pending input for inputId: {}", msg.getInputId());
                       }
                   }
               }
               default -> LOG.warn("Unknown message type: {}", msg.getType());
           }
       });

       ws.onClose(ctx -> {
           String gameId = ctx.pathParam("gameId");
           LOG.info("WebSocket closed for game: {}", gameId);
           GameSession session = activeSessions.remove(gameId);
           if (session != null) {
               session.close();
           }
       });

       ws.onError(ctx -> {
           String gameId = ctx.pathParam("gameId");
           LOG.error("WebSocket error for game: {}", gameId, ctx.error());
           GameSession session = activeSessions.remove(gameId);
           if (session != null) {
               session.close();
           }
       });
   });
   ```

   **handleStartGame method:**
   ```java
   private static void handleStartGame(WsContext ctx, String gameId, InboundMessage msg) {
       WebInputBridge inputBridge = new WebInputBridge();
       ViewRegistry viewRegistry = new ViewRegistry();
       WebGuiGame webGui = new WebGuiGame(ctx, objectMapper, inputBridge, viewRegistry);

       // Use a default AI deck -- for now pick any constructed deck from Forge's data
       // The START_GAME payload can optionally include deckName for human player
       RegisteredPlayer humanPlayer = new RegisteredPlayer(getDefaultDeck())
           .setPlayer(GamePlayerUtil.getGuiPlayer());
       RegisteredPlayer aiPlayer = new RegisteredPlayer(getDefaultAiDeck())
           .setPlayer(GamePlayerUtil.createAiPlayer());

       List<RegisteredPlayer> players = List.of(humanPlayer, aiPlayer);
       Map<RegisteredPlayer, IGuiGame> guis = Map.of(humanPlayer, webGui);

       HostedMatch hostedMatch = new HostedMatch();
       GameSession session = new GameSession(hostedMatch, webGui, inputBridge, viewRegistry, ctx);
       activeSessions.put(gameId, session);

       // Start match on game thread (NOT WebSocket thread)
       ThreadUtil.invokeInGameThread(() -> {
           hostedMatch.startMatch(GameType.Constructed, null, players, guis, null);
       });
   }
   ```
