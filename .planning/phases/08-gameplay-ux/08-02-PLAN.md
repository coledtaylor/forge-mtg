**2. GameBoard.tsx - Add keyboard shortcuts:**

Add import: `import { useHotkeys } from 'react-hotkeys-hook'`

Add keyboard shortcut hooks inside the GameBoard component, AFTER the existing store selectors:

```tsx
// Pass priority / Confirm action - Space or Enter
// Only when no prompt active and buttons enabled (pass priority)
// OR when prompt active and buttons enabled (confirm action)
useHotkeys('space, enter', () => {
  wsRef.current?.sendButtonOk()
}, { enabled: buttons !== null && buttons.enable1 && prompt?.type !== 'PROMPT_CHOICE' })
// Disable during PROMPT_CHOICE because those need specific choice selection, not OK

// Cancel - Escape
useHotkeys('escape', () => {
  wsRef.current?.sendButtonCancel()
}, { enabled: buttons !== null && buttons.enable2 })
```

Note: Do NOT bind shortcuts for PROMPT_CHOICE (the user must click a specific choice). Do NOT bind for PROMPT_AMOUNT (the user must enter a number). But DO bind for the basic button OK/Cancel since those map to pass priority and cancel.

Check if the `wsRef` is stable enough for the callback (it's a ref, so it should be). The `enabled` option from react-hotkeys-hook respects the condition check.
