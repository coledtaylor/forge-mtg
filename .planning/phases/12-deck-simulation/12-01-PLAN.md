    1. **SimulationJob.java** -- Tracks one simulation run's lifecycle:
       - Fields: String id (UUID), String testDeckName, List<String> opponentDeckNames, int totalGames, AtomicInteger completedGames, List<SimulationResult> results (synchronized list), volatile boolean cancelled, volatile boolean running, List<Consumer<SimulationSummary>> progressListeners (copy-on-write)
       - Constructor: (String id, String testDeckName, List<String> opponentDeckNames, int totalGames)
       - addResult(SimulationResult r): adds to results, increments completedGames, notifies listeners with current summary snapshot
       - cancel(): sets cancelled=true
       - isCancelled()/isRunning()/isComplete(): status checks
       - getProgress(): returns current SimulationSummary.computeFrom(results, totalGames) with cancelled flag
       - addProgressListener(Consumer<SimulationSummary>): registers listener
       - removeProgressListener(Consumer): removes listener
       - setRunning(boolean), setComplete(): lifecycle transitions

    2. **SimulationRunner.java** -- Orchestrates simulation execution:
       - Static single-thread executor: Executors.newSingleThreadExecutor with daemon thread named "Simulation-Orchestrator"
       - Static ConcurrentHashMap<String, SimulationJob> activeJobs
       - static SimulationJob startSimulation(String testDeckName, Deck testDeck, Map<String, Deck> opponentDecks, int totalGames):
         a. Create SimulationJob with UUID.randomUUID().toString()
         b. Store in activeJobs
         c. Submit to executor: runSimulation(job, testDeck, opponentDecks)
         d. Return the job
       - private static void runSimulation(SimulationJob job, Deck testDeck, Map<String, Deck> opponentDecks):
         a. Distribute games evenly across opponents (totalGames / opponentCount per opponent, remainder to first)
         b. Alternate play/draw per opponent
         c. For each game:
            - Check job.isCancelled(), break if true
            - Build GameRules: new GameRules(GameType.Constructed), setPlayForAnte(false), setManaBurn(false), setGamesPerMatch(1)
            - Create RegisteredPlayer for test deck and opponent deck
            - Set player via GamePlayerUtil.createAiPlayer(name, "Reckless") for both
            - Create empty guis map: Map.of()
            - Create HostedMatch, call startMatch(rules, null, players, guis, null)
            - Wait for game completion (the startMatch call blocks until the match ends since gamesPerMatch=1)
            - Extract result via GameStatExtractor.extract()
            - job.addResult(result)
            - Null out HostedMatch reference for GC
         d. Mark job complete
         e. Log total time
       - static SimulationJob getJob(String id): lookup from activeJobs
       - static void cancelJob(String id): find and cancel
       - Game timeout: set max 200 turns. After match completes, if lastTurnNumber >= 200, mark as draw.
       - Important: Do NOT use HostedMatch.getDefaultRules() (FModel dependency). Build GameRules explicitly.
       - Important: Both players MUST be LobbyPlayerAi via GamePlayerUtil.createAiPlayer (never use getGuiPlayer()).
  </action>
